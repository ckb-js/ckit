# ckit

[![Github CI](https://github.com/homura/ckit/actions/workflows/ci.yml/badge.svg)](https://github.com/homura/ckit/actions/workflows/ci.yml)

CKB kit

## Development

### Requirement

- Python 3.10
- NodeJS 12.x
- yarn 1.x

### Start Development UI

```
yarn install
yarn build:lib
yarn workspace ckit-app start
```
